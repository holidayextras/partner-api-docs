# Lookup Booking

[API Docs](/hxapi/) > product:[Car Hire](/hxapi/carhire) > endpoint:[lookup booking](/hxapi/carhire/booking/lookup)

```html
GET: /carHire-booking/YourBookingRef
```

## Request

The following GET parameters are mandatory:

```javascript
"key": 1,
"Email": "example@email.com", // the email address that made the booking
"ABTANumber": "AgentCode",
"Password": "xxxxxxxx",
"token": "123456789" // user token
```
