Holiday Extras API

Region Codes

Five or six character region codes determine which region of the world an insurance policy will cover. It is safer to confirm which country your customer is travelling to and map it to the region than ask them the region directly.

Endpoints

Action Endpoint Method
List regions http://api.holidayextras.co.uk/v1/insurancecountries/ GET
List countries for region http://api.holidayextras.co.uk/v1/insurancecountries/foo GET
Search by country http://api.holidayextras.co.uk/v1/insurancecountries/search GET
